> បច្ចុប្បន្នភាពចុងក្រោយសម្រាប់ KhmerGPT

### Introducing GPT-4 Turbo and more (Dec 10, 2023)

Introduce to GPT-4 Turbo the latest generation model by OpenAI. It's more capable, has an updated knowledge cutoff of April 2023.
Available as an Early Access Feature for Plus user.

**And also other features update for all user:**

**Account**

- Forget password

**Setting**

- System language (khmer,english)

**Other**

- Support markdown output 
- Copy To Clipboard 


### Add theme, signup with email (Nov 22, 2023)

Now user can signup with username and password but also with email. 
Reason for this is that we can hanlde better with user security and increased limit daily new users.

**Setting Features:**

- Theme Dark mode and Light mode


### Add User Settings [username,password] (Oct 27, 2023)

**Setting Features:**

- Change Username
- Change Password

**Other Feature:**

- Model switcher GPT-3.5 16k , GPT-3.5

### សូមណែនាំដល់ KhmerGPT Plus (Oct 10, 2023)

យើងបានចេញផ្សាយកំណែថ្មីនៃ KhmerGPT Plus ។ កំណែថ្មីនេះត្រូវបានបង្កើតឡើងនៅលើកំពូលនៃកំណែចុងក្រោយបំផុតនៃម៉ូដែល GPT-3.5 16k ។

**អត្ថប្រយោជន៍នៃ Plus version៖**

- សួរសំណួរ ឬ ជជែកគ្មានដែនកំណត់
- ល្បឿនកាន់តែប្រសើរ
- Model GPT-3.5 16k ( 4 ដងច្រើនជាងបរិបទ GPT-3.5)
- ចូលប្រើមុខងារថ្មីៗមុនគេ

### ការធ្វើបច្ចុប្បន្នភាពការកែលម្អ (March 12, 2023)

**KhmerGPT ត្រូវបានអាប់ដេតសេវាកម្មផ្នែកខាងក្រោយ**

- ធ្វើឱ្យប្រសើរឡើងនៃការឆ្លើយប្រធានបទស្មុគស្មាញs
- ធ្វើឱ្យប្រសើរឡើងនូវការអនុវត្តទូទៅ
- កាត់បន្ថយការជជែកបរាជ័យ 90% ប្រសើរជាងមុន។

### ធ្វើបច្ចុប្បន្នភាពទៅ gpt-3.5-turbo (March 11, 2023)

KhmerGPT ត្រូវបានអាប់ដេតជាមួយនឹងម៉ូឌែលចុងក្រោយបំផុត gpt-3.5-turbo ដែលមានសមត្ថភាពចងចាំការសន្ទនាទាំងមូល និងផ្តល់ចម្លើយកាន់តែប្រសើរ។

**ផ្លាស់ប្តូរ៖**

- ការជម្រុញប្រចាំថ្ងៃរបស់អ្នកប្រើប្រាស់ត្រូវបានកំណត់ឡើងវិញដោយកាត់បន្ថយមកត្រឹម 10 ប្រអប់បញ្ចូលក្នុងមួយថ្ងៃ។
